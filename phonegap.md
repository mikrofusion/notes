
install phonegap
```
npm install -g phonegap
```
install ios specific dependancies
```
npm install -g ios-sim
npm install -g ios-deploy
```

create an app
```
phonegap create com.<companyname>.<appname>
```

build an app
```
phonegap build <ios/andriod/etc>
```

run an app
```
phonegap run --device ios
phonegap run -d ios
phonegap run ios
```
